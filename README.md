# Algorithm Visualizer

An interactive and beginner-friendly visualizer for sorting algorithms (Bubble, Quick, Merge) and Dijkstra's algorithm. Built with **React.js**, **TypeScript**, and **Tailwind CSS**.

## Supported Features

- **Sorting Algorithms**: Visualize Bubble Sort, Quick Sort, and Merge Sort step by step.
- **Graph Algorithm**: Visualize Dijkstra's shortest-path algorithm on a sample weighted graph.
- **Array Size Control**: Adjust the number of bars to be sorted using a slider.
- **Bar Height Control**: Set the maximum height of bars.
- **Speed Control**: Change playback speed for sorting and graph animations.
- **Sorting Order Toggle**: Choose ascending or descending sorting.

## Getting Started
```bash
git clone https://github.com/your-username/algorithm-visualizer.git
cd algorithm-visualizer
npm install
npm start
```

## Tech Stack
- **React.js**
- **TypeScript**
- **Tailwind CSS**
- **React Testing Library + Jest**
